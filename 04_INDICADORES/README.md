# Indicadores e Dados

Fichas para numeros, metricas, faixas, tendencias e evidencias quantitativas.

Exemplos:

- numero de empresas atendidas
- investimento estimado
- maturidade tecnologica
- prioridades por setor
- riscos recorrentes

## Arquivos existentes

- `template-indicador.md` - modelo base para novas fichas de indicador
- `resumo-quantitativo.md` - visão geral do acervo
- `volume-por-setor.md` - distribuição por setor
- `volume-por-territorio.md` - distribuição por território
