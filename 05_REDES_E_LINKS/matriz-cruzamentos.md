# Matriz de Cruzamentos

| eixo | exemplos | uso |
|---|---|---|
| setor x territorio | construção civil x MG/CE/PR | comparar padrões regionais |
| setor x metodologia | inovação x gestão tecnológica x mapa estratégico | ver evolução do formato |
| tema x ano | economia circular x 2024/2025 | ver amadurecimento do tema |
| tema x setor | benchmarking x moda, construção civil, madeira e móveis | encontrar recorrências setoriais |
| territorio x governanca | Curitiba, Santa Catarina, Rio de Janeiro | mapear rede institucional |
| dados x recomendacoes | investimento, demanda, propensão | separar evidência de decisão |
