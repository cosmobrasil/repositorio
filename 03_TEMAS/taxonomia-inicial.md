## Regra de uso

- Cada ficha deve ter uma tag de setor.
- Cada ficha deve ter uma tag de territorio.
- Cada ficha deve ter pelo menos uma tag tematica.
- Dados numericos devem virar fichas separadas em `04_INDICADORES`.
- Recomendacoes recorrentes devem virar fichas em `03_TEMAS`.
