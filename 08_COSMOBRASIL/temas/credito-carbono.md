# Credito de Carbono

Tema de apoio para a base Cosmobrasil.

## Resumo

O material recebido mostra uma taxonomia visual de tipos de credito de carbono. O valor analitico para a base Cosmobrasil esta em conectar decarbonizacao, financiamento, residuos, energia e competitividade territorial.

## Eixos de leitura

- descarbonizacao
- financiamento verde
- eficiencia energetica
- residuos
- agricultura
- energia renovavel
- industria
- territorialidade

## Relevancia para Cosmobrasil

- pode apoiar analises de transicao industrial
- pode apoiar estrategias de circularidade em distritos industriais
- pode ajudar a formular instrumentos economicos para projetos territoriais
