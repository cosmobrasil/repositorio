# Processo de Ingestao Cosmobrasil

Fluxo padrao para incorporar novas fontes ao acervo Cosmobrasil.

## 1. Receber a fonte

- caminho local
- link
- texto colado
- arquivo anexado
- referencia bibliografica

## 2. Registrar no indice

- atribuir id
- registrar origem
- classificar tipo
- apontar data e autor, quando houver
- marcar status inicial

## 3. Ler e extrair

- resumo executivo
- conceitos centrais
- atores relevantes
- evidencias numericas
- implicacoes para economia circular
- implicacoes para distritos industriais

## 4. Classificar

- ficha
- tema
- indicador
- rede
- sintese
- material bruto

## 5. Conectar

- ligar a fichas existentes
- ligar a temas recorrentes
- apontar cruzamentos com outros documentos
- registrar lacunas de informacao

## 6. Consolidar

- atualizar a sintese Cosmobrasil
- atualizar a visao do agente especialista
- revisar status da fonte

## Status sugeridos

- `pendente`
- `recebido`
- `em_leitura`
- `sintetizado`
- `validado`
