# Protocolo de Validacao Cosmobrasil

Regras para validar o material que entra na base Cosmobrasil.

## 1. Registro de origem

- confirmar de onde veio o material
- registrar tipo de fonte
- registrar data, quando houver
- registrar autor, quando houver
- registrar caminho ou link original

## 2. Leitura minima

- identificar assunto principal
- identificar atores citados
- identificar territorio citado
- identificar relacao com Cosmobrasil
- identificar relacao com economia circular
- identificar relacao com distritos industriais

## 3. Classificacao

- `bruto` quando a fonte ainda nao foi lida
- `recebido` quando a fonte foi registrada
- `em_leitura` quando o conteudo esta sendo extraido
- `sintetizado` quando a leitura virou ficha ou sintese
- `validado` quando a fonte foi cruzada com outra evidencia

## 4. Critérios de validacao

- consistencia interna
- rastreabilidade da origem
- clareza do contexto
- compatibilidade com outras fontes
- relevancia para o caso Cosmobrasil

## 5. Tratamento de conflito

- registrar o conflito explicitamente
- preservar as duas versoes quando nao houver evidencia decisiva
- indicar qual fonte tem maior peso
- evitar conclusao forte sem triangulacao

## 6. Regra de ouro

Se a fonte nao puder ser confirmada, ela pode entrar como evidencia provisoria, mas nao deve ser tratada como fato consolidado.
